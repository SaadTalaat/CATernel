00007 /* HOW GDT register is used
00008  * This is the GDT register
00009  * ---------------------------------------------
00010  *|31                 16| 15                   0|
00011  *|                     |                       |
00012  *|     Base 0:15       |       Limit 0:15      |
00013  *|                     |                       |
00014  *|---------------------------------------------|
00015  *|63   56|55 52| 51  48|47      40|39        32|
00016  *| Base  |     | Limit | Access   |            |
00017  *| 24:31 |Flags| 16:19 | Byte     | Base 16:23 |
00018  *|       |     |       |          |            |
00019  * ---------------------------------------------
00020  * Access byte is
00021  * byte 0 = Accessed bit set to 1 by CPU when segment is accessed. we will set it to 0
00022  * byte 1 = read/write permissions 
00023  * byte 2 = Direction bit we will set that to 0 for growing up segments and 1 for growing down segments and conforming bit
00024  * byte 3 = Executable bit 1 if code segment 0 if data segment
00025  * byte 4 = always 1
00026  * byte 5,6 = Privilege since we are a kernel we will set that to 0
00027  * byte 7 = Present bit one for anything
00028  *
00029  * Flag 4 bits are always 1100
00030  */
